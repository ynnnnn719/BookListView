BookListView

這是一個模擬圖書借閱系統，他建立了一個書籍清單介面，系統內預先加入多本經典文學作品，包含書名、作者與書籍類別。使用者可以透過介面瀏覽書籍資訊，並切換不同的顯示模式來查看內容。

除了基本的書籍顯示功能之外，也加入了簡易的借閱流程。當使用者點選書籍後，系統會詢問是否借閱，確認後便會將該書加入借閱清單中，同時避免重複借閱相同書籍。

書籍借閱功能
當使用者選擇一本書時：

系統會確認是否借閱
若確認借閱
將書籍加入借閱清單
若已借閱過則不重複加入。
